DriverStation

This page includes information and links (see More Information) about the FRC Driver Station.

General support questions should be directed to driverstation_at_kwikbyte.com

Technical Specs

  • RoHS Compliant
  • Atmel AT91SAM9260 processor
  • 200 MHz, ARM926EJ-S core with Java acceleration and DSP instruction extensions
  • Independent 8KB instruction and 8KB data caches
  • 64 MB SDRAM
  • 8 MB boot/kernel Flash
  • 1 GB NAND Flash - not installed
  • 2 x 10/100 Ethernet ports
  • 4 x USB 2.0 full speed host ports
  • 1 x competition port
  • 8 x digital input channels (+5VDC in user header)
  • 8 x digital output channels (+5VDC in user header)
  • 4 x analog input channels (+5VDC range in user header)
  • 3 x user buttons
  • 1 x mode toggle switch
  • 1 x LCD monochrome graphic (128x64)
  • Sturdy metal case
